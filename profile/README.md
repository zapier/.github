# Zapier

API infrastructure for AI agents — authenticated, governed access to 9,000+ apps.

## Quick Start

```bash
npm install @zapier/sdk    # build agent-powered automations in code
```
```bash
npx @zapier/mcp            # give AI apps access to 9,000+ apps instantly
```

## Primitives

- [Zapier SDK](https://zapier.com/sdk) — Authenticated, governed access to the full Zapier catalog in code, on behalf of your users. Includes a CLI for managing connections and testing from the terminal.
- [Zapier MCP](https://zapier.com/mcp) — Connect AI apps like Claude, ChatGPT, and Cursor to 9,000+ apps through a single MCP server.
